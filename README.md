# 맛따라

전라남도 구례군의 맛집과 명소를 발견하고 구례 안에서 하루 코스를 추천받는 React 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

전체 검증은 `npm run check`로 실행합니다. 이 명령은 파일별 실코드 줄 수, ESLint,
단위·DOM 테스트, 프로덕션 빌드를 차례로 확인합니다.

## 주요 화면

- 홈: 관리 데이터 기반 카테고리 필터와 맛집 피드
- 상세: 사진, 장소 정보, 지도, 길찾기
- 추천 코스: 세 가지 코스와 단계별 지도 동선
- 카테고리 관리: 카테고리 추가·수정·삭제와 기본값 복원
- 로그인/회원가입: 아이디·비밀번호, 세션 유지, Kakao·Google OAuth 진입
- 마이페이지: 사용자별 게시물 북마크 저장과 모아보기
- 게시물 공유: 링크 복사, 기기 공유, X·Facebook 공유
- 맞춤 추천: 조회·북마크에서 계산한 태그 및 지역 선호도를 쿠키에 저장해 추천
- 구례 태그 추천: 산수유·릴랙스·로컬푸드·한옥카페·자연 선택 가중치를 localStorage에 저장
- 구례 전용 데이터: 홈·검색·맞춤 추천·추천 코스에서 구례군 밖의 장소를 차단
- 이미지 캐러셀: 피드 자동 전환, 화살표·모바일 스와이프, 상세 사진별 코멘트 연동
- 탐색 UX: 내부 이동 기록 기반 뒤로가기와 좌우 분할 코스 지도·하단 이동 순서
- 맛집 등록: 로그인한 사용자의 검수 요청 폼

## 로그인과 제출 저장소

Supabase를 연결하지 않으면 아이디·비밀번호 계정과 세션이 현재 브라우저에 저장되는
로컬 체험 모드로 동작합니다. 비밀번호는 PBKDF2 해시로 저장되지만, 실제 서비스에서는
반드시 서버 인증을 사용해야 합니다.

Supabase 프로젝트를 연결하면 이메일·비밀번호 인증과 세션 유지가 Supabase Auth로
자동 전환됩니다. Kakao와 Google 공급자를 활성화한 뒤 `.env.example`을 `.env`로
복사하고 실제 값을 입력합니다. SQL 편집기에서 `supabase/schema.sql`을 실행하면 로그인
사용자가 자신의 맛집 등록 요청을 생성하고 조회할 수 있는 테이블과 RLS 정책이 설정됩니다.

카테고리 CRUD는 현재 프로토타입 범위에서 브라우저 `localStorage`에 저장됩니다. 운영
전환 시 관리자 권한을 검증하는 서버 API 또는 Supabase 테이블로 저장 계층을 교체해야
합니다.

## 모듈 기준

빈 줄과 주석을 제외해 파일당 250줄을 소프트 리미트, 400줄을 하드 리미트로 적용합니다.
`scripts/check-lines.mjs`가 이 기준을 자동 검사합니다.
